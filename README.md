# 简介
利用github action 实现自动健康打卡


## 写在前面
钉钉机器人推送代码来自[Dimlitter](https://github.com/Dimlitter)同学


## 需要的secrets`必填`
```
account:通行证账号
 
password:通行证密码
```

## 可选参数`推送相关`
```
DD_BOT_TOKEN: 钉钉自定义机器人的token，只需 https://oapi.dingtalk.com/robot/send?access_token=XXX 等于=符号后面的XXX即可

DD_BOT_SECRET ：钉钉机器人的加签密钥
```
